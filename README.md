# Igiwa - Chip8 Emulator

[CHIP-8](https://en.wikipedia.org/wiki/CHIP-8) Emulator written with C

## Build

Use **make** to to build this project

```bash
make
```

## Clean

The build directory can be cleaned with:

```bash
make clean
```

## Run the VM

Start the CHIP-8 Emulator with a Rom

```bash
cd build
./chip8 name_of_rom
```
